## 🐛 Troubleshooting

### GPU not detected?
```powershell
python -c "import tensorflow as tf; print(tf.config.list_physical_devices('GPU'))"
```

### Out of memory?
```python
# In train_tflite_model.py:
self.BATCH_SIZE = 8          # Reduce batch size
self.IMG_SIZE = 224          # Reduce image size
```

### Model training failed?
```powershell
# Check dataset is properly formatted
python verify_setup.py
```

### Poor accuracy on camera?
- Check lighting conditions match training data
- Retrain with more epochs
- Use larger IMG_SIZE during training

- Save model checkpoint before stopping training
- Test model locally before deploying to camera
